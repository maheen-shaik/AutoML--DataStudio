# AutoML--DataStudio
Think of it as your AI data assistant. You upload messy data, describe what you want in plain English or speak it, and it gives you a clean, ready to use dataset along with insights and code.

What this project actually does

Building ML models is interesting, but cleaning data is where most of the time is spent.

AutoML Data Studio solves that problem.

Instead of manually writing preprocessing code, you just:

Upload your dataset
Tell the system what you want such as remove nulls and scale features
Get a clean dataset, visual insights, and ready to use Python code
Features in simple words

Authentication
Create an account and securely access your projects.

Voice Control
Speak your query instead of typing, such as
Remove outliers and show correlation

Multi format Upload
Upload files in CSV, Excel, or JSON formats.

Dataset Merging
Upload multiple datasets and the system automatically merges them using common columns.

Natural Language Queries
No coding required. Example
Fill missing values and normalize data

Visualization Dashboard
Instant charts including
Histograms
Boxplots
Heatmaps
Pairplots

Code Generator
Get a clean Python script that you can directly use in your project.

CSV Export
Download the processed dataset with one click.

Why this is useful

Saves time spent on data preprocessing
Makes data work accessible to non programmers
Useful for students, hackathons, and rapid ML prototyping
Combines AI, data science, and full stack development

Tech Stack simple explanation

Python and FastAPI for the backend
React and Vite for the frontend
PostgreSQL for database management
Web Speech API for voice interaction
